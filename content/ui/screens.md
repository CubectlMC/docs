# Экраны web-интерфейса

## Auth

Содержит:

- Login по никнейму и паролю
- Регистрацию по никнейму и паролю
- Logout
- Refresh токена
- Профиль текущего пользователя

## Панель управления

Содержит:

- Список инстансов
- Агрегированный статус
- Суммарное использование RAM
- Суммарное использование CPU
- Предупреждения по лимитам
- Последние события

## Инстансы

Содержит:

- Список инстансов
- Фильтр по статусу
- Создание инстанса
- Запуск
- Остановку
- Перезапуск
- Переход в карточку инстанса

## Создание инстанса

`POST /instances` создает полный профиль инстанса

Экран содержит:

- Версию
- Loader type и loader version
- Java version mode
- Memory limit
- CPU limit
- Server settings
- Selected mods
- Selected plugins
- Selected datapacks
- Network и DNS settings
- Mounted `/data` directory
- Docker container
- Optional start after create

## Карточка инстанса

Содержит вкладки:

- Обзор
- Среда выполнения
- Контент
- Логи
- Метрики
- Участники
- События

## Контент инстанса

Использует:

- `GET /instances/{instance_id}/content`
- `PATCH /instances/{instance_id}/content`

Контент включает:

- Моды
- Плагины
- Дата-паки

Контент выбирается через provider id, project id и version id

Изменение контента разрешено только когда инстанс остановлен

## Каталог контента

Содержит:

- Поиск модов
- Поиск плагинов
- Поиск дата-паков
- Фильтр по версии и loader
- Карточку проекта
- Версии проекта
- Зависимости
- Статус провайдера контента
- Проверка обновлений
- Кнопка обновления доступной версии

## Метрики

Содержит:

- Текущие метрики инстанса
- Исторические метрики инстанса
- Live-метрики инстанса через SSE
- Текущие метрики хоста
- Исторические метрики хоста
- Live-метрики хоста через SSE
- CPU
- RAM
- Disk
- Network
- Uptime

## Логи

Содержит:

- Живые Docker-логи через SSE

## Пользователи и доступы

Содержит:

- Пользователей
- Роли
- Permissions
- Глобальные роли пользователя
- Участников инстанса
- Выдачу доступа к чужому инстансу

## Лимиты ресурсов

Админский экран

Содержит:

- Общий RAM budget
- Свободную RAM на хосте
- Системный RAM reserve
- Минимальный RAM на инстанс
- Максимальный RAM на инстанс
- CPU policy
- Диапазон портов
- Максимальное число запущенных инстансов
